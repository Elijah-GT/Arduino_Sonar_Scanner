# Arduino_Sonar_Scanner
Arduino based sonar scanner. Uses ultrasonic sensor to detect objects and has varying alert levels depending on distance.
Components:
    -Arduino UNO
    -Ultrasonic Sensor
    -Servo Motor SG90
    -Passive Buzzer
    -RGB LED
    -Three 220Ohm Resistors
    -Hot Glue Gun
    -Masking Tape
